Tekken Framedata Bot

This Discord bot provides frame data for a given move of a specified character in Tekken 8 and Tekken 7. Perfect for mastering your gameplay!

Add to Server

How to Use

Follow these simple steps to get started:

  • Invite the bot using the button above.
  • Try out commands like /fd8 or /fd7.

Commands

  • Framedata (Tekken 8): /fd8 {character_name} {attack_input_notation}
    Example: /fd8 reina df2
  • Framedata (Tekken 7): /fd7 {character_name} {attack_input_notation}
    Example: /fd7 bryan df23
  • Ping: /ping - Replies with "Pong!"
  • Help: /help - Lists all commands
  • Report: /report {message} - Send feedback
  • Support: /support - Support information
